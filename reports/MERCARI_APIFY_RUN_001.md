## Troubleshooting

### If Run Fails
**Check:**
1. APIFY_TOKEN set correctly
2. Actor still works (test with category/84)
3. Network/API availability
4. Output directory permissions

### If Zero Items
**Possible Causes:**
1. No active listings for search terms
2. Mercari blocking requests (enable proxy)
3. Actor timeout
4. Search terms too specific

### If Cost Exceeds Budget
**Actions:**
1. Reduce searches (fewer models)
2. Lower max_items_per_url
3. Run less frequently
