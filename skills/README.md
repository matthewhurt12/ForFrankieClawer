# Local Skills

This folder contains reusable OpenClaw/Frankie skills for this mind.

## Available

- `page-map` - map interactive elements on a web page before browser automation.
- `arbitrage-deal-desk` - run and explain the retro tech arbitrage pipeline.
- `signals` - full Signal Engine report for `/signals`.
- `radar` - short current scan for `/radar`.
- `bigpic` - strategic synthesis for `/bigpic`.
- `ideas` - action and opportunity report for `/ideas`.

## Discovery

If a skill is not found automatically, use:

- `COMMANDS.md`
- `skills/manifest.json`
- `.openclaw/skill-manifest.json`

Each skill folder contains a `SKILL.md` file. Open that file before running the
workflow.

Read each skill's `SKILL.md` before running it.
